Haqqında


 

Saytın ilk versiyası olan aligoshgar.cf saytı Əliqoşqar Kərimli tərəfindən 10 aprel 2013-cü ildə yaradılmışdır. 21 dekabr 2017-ci ildə domen dəyişikliyinə gedilmiş və saytın domen uzantısı .ga olaraq dəyişdirilmişdir. 03 iyul 2023-cü ildə isə saytın serveri olan Freenom-da baş verən texniki xəta səbəbi ilə .ga domeni ləğv edilmiş və sayt öz fəaliyyətini .cf domen uzantısı ilə davam etdirmişdir. 03.09.2023-cü il tarixdə Freenom-un təqdim etdiyi bütün domen adları Əliqoşqar Kərimli tərəfindən ləğv edilmiş və sayt Github platformasında fəaliyyətinə davam etmişdir.Əvvəllər özündə müxtəlif xidmətləri, müəllifin müxtəlif vaxtlarda və çox müxtəlif situasiyaların təsirilə yazdığı məqalələri, müəllifə məxsus məlumatları, şəkilləri, videoları da birləşdirsə də, bu tarixdən sonra saytın istifadə təyinatı dəyişdirilərək ödənişsiz xidmətlər və ödənişsiz IPTV platformasına çevrilmişdir. Ödənişsiz IPTV xidmətilə bağlı ətraflı məlumat saytın IPTV səhifəsində mövcuddur. Saytın əvvəllər təqdim etdiyi digər xidmətlər isə Əsas səhifədə təqdim edilməyə davam edir! Bildirmək istəyirik ki, Blogger vasitəsilə kodlanıb CNAME Records kodları vasitəsilə Freenom-da fəaliyyət göstərən saytımızın Freenom-a aid heç bir domain ilə (.tk ; .ml ; .ga ; .cf ; .gq) hər hansı bir əlaqəsi qalmamışdır! Belə ki, sayt Əliqoşqar Kərimli tərəfindən Github platformasında ən başdan kodlanıb hazırlanmışdır. Əliqoşqar Kərimli qaynaq kodları qapalı olan bir çox platforma olmasına rəğmən saytı bilərəkdən qaynaq kodları açıq olan Github platformasında kodlayıb hazırlamışdır. Ümid edirik ki, bütün bu dəyişikliklər hər bir istifadəçimizin ürəyincə olacaqdır! Hörmətlə, aligoshgar.github.io idarə heyəti.

Youtube ReVanced


Premium funksiyalara sahib - reklamsız və arxa planda istifadəsi mümkün olan YouTube axtarışındasınızsa, sizlərə tam ödənişsiz Youtube ReVanced-i təqdim edirik! Bildiyiniz kimi YouTube Vanced öz fəaliyyətinə rəsmi şəkildə son verib. Lakin ReVanced bu eyni funksiyalarla onun fəaliyyətini davam etdirir! YouTube bütün keçmiş Premium modifikasiyalı versiyalarına qadağa qoyduğu üçün bu yazımızda sizə ReVanced Manager tədbiqi vasitəsilə yeni versiyanı əldə etməyin izahını verəcəyik. Bunun üçün YouTube-un v19.11.43 versiyası üzərində ReVanced Manager tədbiqi vasitəsilə müvafiq modifikasiyalar edilməlidir. Lakin edəcəyiniz modifikasiya əsnasında üzləşə biləcəyiniz çətinlikləri nəzərə alaraq sizə revanced.to saytından alınan, hazır modifikasiya edilmiş YouTube Revanced v19.11.43 və ReVanced-in Github səhifəsindən alınan, hesabınıza girişinizi, eləcə də proqramın işləkliyini təmin edən köməkçi proqram GmsCore MicroG Services tədbiqini təqdim edirik: 1. YouTube ReVanced v19.46.42; 2. GmsCore MicroG Services; Bu tədbiqləri endirin və quraşdırın. GmsCore MicroG Services tədbiqini yükləmədən YouTube-a giriş etdiyiniz təqdirdə proqram sizi bu tədbiqin yükləmə səhifəsinə, yəni ReVanced Manager-in rəsmi Github səhifəsinə yönləndirəcəkdir. Hər iki tədbiqi yükləyib quraşdırdıqdan sonra edəcəyiniz son 1 əməliyyat qalır! Quraşdırdığınız GmsCore MicroG Services tədbiqinə giriş edin. Ən yuxarıdakı Self-Check bölməsinə giriş edin. Açılan səhifənin ən sonuna gələrək Battery optimizations ignored seçiminin qarşısındakı qutucuğu işarələyin. Proqram sizi avtomatik surətdə cihazınızın modelindən asılı olaraq Batareyaya qənaət rejimində arxa planda işləyə bilən proqramların olduğu ayarlar bölməsinə yönləndirəcəkdir.
