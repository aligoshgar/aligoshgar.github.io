KMS Auto

KMS nədir?


KMS Auto proqramı Windows üçün hazırlanmış olan, Windowsun və Microsoft Office 365-in (Word və Excell kimi bütün ofis proqramları) lisenziya (product key) tələb edən ödənişli versiyalarının tam ödənişsiz istifadəsinə imkan yaradan proqramdır. Aşağıdakı link vasitəsilə proqramı yükləyin və hər hansı ZIP arxiv proqramı vasitəsilə rsload şifrəsini daxil edərək lazımi faylları çıxarıb quraşdırın. Proqramın istifadəsi asandır. Giriş edərkən sizi birbaşa aktivləşdirmə butonları qarşılayacaqdır. Yükləmə linki: KMSAuto++ 1.8.7


Aktivləşdirmə sonrasında ekranın sağ aşağı küncündə hələ də "Activate Windows" yazısı görükürsə aşağıdakı addımları tədbiq edin:

1. Wındows axtarış qisminə "Run" yazın və tapılan proqramı başladın;
2. Açılan proqramda regedit yazaraq OK butonuna basın;
3. Ekrana gələ biləcək hər hansı bir xəbərdarlığa "bəli" seçərək davam edin;
4. Açılan qovluqların arasında bu ardıcıllıq ilə irəliləyin:
HKEY_LOCAL_MACHINE > SYSTEM > CurrentControlSet > Services > svsvc
5. Bu qovluqda olan Start faylının üstünə sağ click edərək ilk sıradakı Modify seçimini edirik;
6. Açılan pəncərədə dəyər göstəricisi 0 və ya 3-dürsə, onu 4 edərək OK butonuna basın;
7. Kompüteri yenidən başladın.
