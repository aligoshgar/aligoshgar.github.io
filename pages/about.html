Öncəliklə onu bildirmək istəyirik ki, aligoshgar.cf saytı Əliqoşqar Kərimli tərəfindən 10 aprel 2013-cü ildə yaradılmışdır. 21 dekabr 2017-ci ildə domen dəyişikliyinə gedilmiş və saytın domen uzantısı .ga olaraq dəyişdirilmişdir. 03 iyul 2023-cü ildə isə saytın serveri olan Freenom-da baş verən texniki xəta səbəbi ilə .ga domeni ləğv edilmiş və sayt öz fəaliyyətini .cf domen uzantısı ilə davam etdirmişdir. 03.09.2023-cü il tarixdə Freenom-un təqdim etdiyi bütün domen adları Əliqoşqar Kərimli tərəfindən ləğv edilmiş və sayt Github platformasında fəaliyyətinə davam etmişdir.Əvvəllər özündə müxtəlif xidmətləri, müəllifin müxtəlif vaxtlarda və çox müxtəlif situasiyaların təsirilə yazdığı məqalələri, müəllifə məxsus məlumatları, şəkilləri, videoları da birləşdirsə də, bu tarixdən sonra saytın istifadə təyinatı dəyişdirilərək ödənişsiz xidmətlər və ödənişsiz IPTV platformasına çevrilmişdir. Ödənişsiz IPTV xidmətilə bağlı ətraflı məlumat saytın IPTV səhifəsində mövcuddur. Saytın əvvəllər təqdim etdiyi digər xidmətlər isə Əsas səhifədə təqdim edilməyə davam edir! Bildirmək istəyirik ki, Blogger vasitəsilə kodlanıb CNAME Records kodları vasitəsilə Freenom-da fəaliyyət göstərən saytımızın Freenom-a aid heç bir domain ilə (.tk ; .ml ; .ga ; .cf ; .gq) hər hansı bir əlaqəsi qalmamışdır! Belə ki, sayt Əliqoşqar Kərimli tərəfindən Github platformasında ən başdan kodlanıb hazırlanmışdır. Əliqoşqar Kərimli qaynaq kodları qapalı olan bir çox platforma olmasına rəğmən saytı bilərəkdən qaynaq kodları açıq olan Github platformasında kodlayıb hazırlamışdır. Ümid edirik ki, bütün bu dəyişikliklər hər bir istifadəçimizin ürəyincə olacaqdır! Hörmətlə, aligoshgar.github.io idarə heyəti.
